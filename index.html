My Recent Projects

Here are a few past projects I've worked on.

Job Card Organization PT Dirgantara Indonesia

A web-based system for PT Dirgantara Indonesia (PTDI) that simplifies employee task tracking, progress monitoring, and work status management. The platform features a clean, minimalist design, a well-structured database, and includes ongoing maintenance and error resolution support.

SUI SOES

A mobile app for Android that serves as an online shoe store, developed using Kodular and Firebase. The app allows users to browse shoe collections, add items to the cart, and place orders. It provides an easy-to-use shopping experience with real-time database integration.

Algorithm Comparison

Web developed for a final-year project that compares the performance of Jaro-Winkler and Levenshtein Distance algorithms in detecting misspelled words. The system demonstrates how each algorithm works, provides correction suggestions, and helps users understand differences in accuracy and effectiveness.

THE NEWS

A web application for publishing and managing news articles, built using Laravel. The platform allows users to read, create, edit, and manage news content, featuring a responsive design and user-friendly interface.

POS (Point of Sale)

POS Web Application using Laravel 12 that can help businesses monitor inventory in real-time. Every time a transaction is made, the POS system automatically updates the number of items available, including calculating the total number of items, receiving payments and returns, printing reports for goods receipts and transactions, and payment receipts/notes, thereby facilitating inventory management.
